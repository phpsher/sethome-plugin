# SetHome Plugin

Простой Minecraft-плагин, позволяющий игрокам сохранять и использовать точки дома.

## Возможности
- Установка дома с именем
- Телепортация в сохранённый дом
- Просмотр списка всех домов игрока

## Команды

| Команда | Описание |
|--------|--------|
| `/sethome <name>` | Установить точку дома |
| `/home <name>` | Телепортироваться в указанный дом |
| `/homes` | Показать список всех домов |

## Требования
- Minecraft Server API **1.21+**
- Java **17+**

## Установка
1. Скомпилируйте плагин в `.jar`
2. Переместите файл в папку `plugins` вашего сервера
3. Перезапустите сервер
